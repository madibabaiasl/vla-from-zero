# From Zero to Vision-Language-Action Models

Welcome. This is a free, fully browser-based course that takes you from "I have never trained a model" to "I fine-tuned a Vision-Language-Action model on a real robot dataset."

You will not install anything. Every lesson opens in Google Colab with one click, and every concept is paired with code you run and tweak as you read.

## What you will build

By the end of this course, you will have:

1. Trained your own small neural networks for images and text.
2. Understood modern robotics math, the same way my graduate robotics course teaches it, using screw theory and the product of exponentials.
3. Controlled a simulated robot arm from a browser notebook.
4. Trained a behavior cloning policy and a small diffusion policy.
5. Built a tiny Vision-Language-Action model from scratch.
6. Fine-tuned an open VLA on a real robot dataset and pushed it to the Hugging Face Hub.

## Who this is for

Total beginners to machine learning and robotics. You only need:

1. A web browser.
2. A free Google account (for Colab).
3. A free Hugging Face account (we will create it together in Module 0).
4. Curiosity and about two evenings per module.

## How to use this site

Every page with a rocket icon at the top can be launched directly in Google Colab. Click the rocket, pick Colab, and the lesson opens as a runnable notebook.
